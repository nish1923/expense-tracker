# Expense Tracker for Students

## Overview

A simple web-based expense tracker designed for students to manage daily spending efficiently.

## Features

* Add and track expenses
* Real-time total calculation
* Simple and clean UI

## Tech Stack

* HTML
* CSS
* JavaScript

## Purpose

This project helps students build financial awareness and manage their expenses effectively.

## Future Improvements

* Add charts for visualization
* Category filtering
* Monthly reports
